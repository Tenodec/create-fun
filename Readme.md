## 使用

```
yarn create fun xxx
or
npm create fun xxx
```

## 说明
### 目前仅支持ts项目
 
### 创建的模板基于vite+react+arco，打造轻量化的后台管理模板
### 更新模板依赖
在创建的项目目录下运行
```
//推荐
yarn upgrade-interactive --latest

```
### 此脚手架借鉴了create-vite
